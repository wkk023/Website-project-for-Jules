# Risk-based Building Fire Safety Visit Programme - 背景與目標

## 計畫目標

1. **增強操作準備度** - 通過熟悉訪問提高消防隊的應急反應能力
2. **解決消防安全違規** - 針對常見區域的實質性消防安全違規
   - 例如：EVA 堵塞、MoE 堵塞、FSI 缺陷
3. **建立結構化領土訪問計畫** - 定期進行有組織的領土範圍訪問
4. **進行主動公共教育** - 進行更積極和直接的公共教育

## 訪問範圍 (Scope of Fire Safety Visit)

### 任務 1：訪前準備工作 (20 分鐘)
- BRAVO - EVA & 建築佈局
- LIFIPS - FS21, FS251, 持牌食肆
- 其他信息 - VRW, 迷你倉, DG 持牌場所

### 任務 2：設備與水供 (5-10 分鐘)
- a) 設備位置 - EVA
- b) 水供 - 街道消防栓、FSI 進水口

### 任務 3：內部消防設備與搜救 (25-40 分鐘)
- a) 內部消防設備
  - 典型樓層佈局
  - 消防員升降機
  - 樓層著陸處的消防栓
- b) 搜救 (SAR)
  - 逃生出口
  - G/F 和屋頂出口

**總計訪問時間：50-70 分鐘**

## 檢查頻率 (需要從 PDF 的其他頁面確認)

根據建築物風險等級，需要確認：
- A 級建築物的檢查頻率
- B 級建築物的檢查頻率
- C 級建築物的檢查頻率
- D 級建築物的檢查頻率
- E 級建築物的檢查頻率

## 關鍵概念

- **熟悉訪問 (Familiarization Visit)** - 允許其他消防隊成員熟悉建築物，無需由同一名消防員進行檢查
- **風險等級 (Risk Category)** - 根據建築物特徵確定的分類
- **特別風險 (Special Risk)** - 如車房、迷你倉、食肆等特殊用途場所


## 風險等級分類決策樹

根據建築物特徵，系統使用決策樹來分配風險等級：

### 工業建築/倉庫
- 完全灑水系統：B 級
- 無灑水系統：A 級

### 商業建築
- 單一樓梯或非部分灑水系統：
  - 建築高度 ≥ 30m：A/A* 級
  - 建築高度 < 30m：B/C 級
- 特別用途（VRW、迷你倉、DG、持牌食肆等）：需要額外考慮

### 住宅建築
- 3 層或以下：E 級
- 超過 3 層：
  - 單一樓梯：B 級
  - 多樓梯且建築高度 ≥ 30m：C 級
  - 多樓梯且建築高度 < 30m：D 級

### 綜合式樓宇
- 單一樓梯：A/B/C 級（取決於特別用途和樓層）
- 多樓梯：C/D 級（取決於特別用途和樓層）

### 機構建築/其他
- 特別用途（醫院、酒店、危險品等）：A*/A/B 級
- 其他：B/C/D 級

## 檢查頻率 (需要從 PDF 確認)

根據風險等級，檢查頻率應如下：
- **A/A* 級**：最高頻率（需要確認具體頻率）
- **B 級**：中高頻率（需要確認具體頻率）
- **C 級**：中等頻率（需要確認具體頻率）
- **D 級**：較低頻率（需要確認具體頻率）
- **E 級**：最低頻率（需要確認具體頻率）

## 新功能需求

基於用戶的要求，系統需要增加以下功能：

1. **檢查後風險等級更新**：消防人員可在檢查後更新建築物的最終風險等級
2. **檢查員選擇**：記錄進行檢查的消防隊編號 (A/B/C)
3. **熟悉訪問**：允許其他消防隊成員熟悉建築物，無需由同一消防員進行檢查
4. **檢查歷史追蹤**：清楚顯示建築物在本年度已檢查的次數
5. **下次檢查預測**：根據風險等級和檢查頻率，計算下次檢查的預定日期
